# Secure Login Analyzer 🔐

Secure Login Analyzer is a project designed to enhance user authentication security.
It provides user registration and login functionality with additional security features
to prevent unauthorized access.

---

## 🚀 Features

- User Registration and Login
- Incorrect username/password detection
- Account lock after 3 failed login attempts
- Security monitoring for suspicious login attempts
- Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript / React
- Backend: Python (Flask) / Java
- Database: MongoDB
- Tools: Git, GitHub

---

## 📂 Project Running method 
cd backend
pip install -r requirements.txt

cd frontend
